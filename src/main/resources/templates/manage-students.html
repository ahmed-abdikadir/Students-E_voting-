CSV Format Guide

                            
Your CSV file should have the following format with a header row:

                            

                                username,fullName,role
student001,John Doe,STUDENT
student002,Jane Smith,STUDENT
admin001,Admin User,ADMIN
                            

                            

                                Notes:
                                
    
                                    
  • Username must be unique (no duplicates)
    • 
                                    
  • Role must be either 'STUDENT' or 'ADMIN'
    • 
                                    
  • Fields are case-insensitive (STUDENT/student both work)
    • 
                                    
  • Duplicate usernames will be skipped
    • 
                                    
  • Default password will be the username
    • 
                                

                            

                        


                        
                        

                            
                            
                            Maximum file size: 5MB
